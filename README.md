# Hockeymanager v2

Tämä versio säilyttää alkuperäisen ulkoasun ja alkuperäisen suuren pelaajalistan.

## Käyttöönotto
1. Kopioi tiedostot GitHub-repoosi.
2. Aja `supabase_schema.sql` Supabase SQL Editorissa.
3. Lisää Railwayhin `.env.example`-tiedoston muuttujat oikeilla arvoilla.
4. Deployaa Railwayhin.

## Mukana
- alkuperäiset HTML-sivut säilytettyinä
- korjattu `main.py`
- lisätty `pelikalenteri.html`
- Supabase-skeema nykyisille snake_case -kentille
